# OPENPAAS-IEDA-WEB

**플랫폼 설치 자동화는 BOSH CLI(command line interface) 실행환경을 웹으로 구현한 오픈 소스이다.**

**플랫폼 설치 자동화의 개발환경은 다음과 같다.**
- Ruby (1.9.3 이상)
- bosh-init
- bosh_cli
- spiff
- Java (1.8 이상)
- maven
- mysql
- go (1.5 이상)
